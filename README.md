Equations have to be in RPN (can only handle single digits)

Examples:

34+ == 3+4 = 7

234^+ == 3^4+2 = 83

34*25*+ == (3*4)+(2+5) = 22

342+*5* == 3*(4+2)*5 = 90

Edit configurations to your working directory to run the program.
